# Закрывающий тег

Учебный проект из курса Яндекс Практикума.

## Описание

Веб-страница с карточками воспоминаний о прохождении курса веб-разработки.

## Технологии

HTML, CSS, JavaScript

## Функциональность

- Адаптивная вёрстка
- Анимация лайка
- CSS-фильтры
- Модальное окно

## Запуск

Открыть `index.html` в браузере.
